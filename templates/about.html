Explore More About Miraculous

Miraculous is an innovative dataset discovery system designed to empower researchers and professionals to efficiently find, explore, and integrate datasets across federated systems. It uses semantic matching, interactive search, and other advanced techniques to redefine dataset discovery.

  • Integration with various data repositories (local, relational, document-based).
  • Semantic search powered by Exhaustive, Approximate, and Clustered Targeted methods.
  • Interactive query refinement based on user preferences.
  • Support for both keyword and table-based queries.

Exhaustive Search

This method thoroughly compares the input query against every cell in every dataset. While it guarantees comprehensive results, it is computationally intensive, making it ideal for scenarios where completeness is crucial.

Approximate Nearest Neighbors

ANN focuses on finding the closest matches to the query by using approximation algorithms. This method strikes a balance between speed and accuracy, making it suitable for large-scale or time-sensitive searches.

Clustered Targeted Search

This method organizes datasets into clusters based on semantic similarity and searches only within the most relevant clusters. It provides both precision and efficiency, particularly for diverse and complex datasets.
